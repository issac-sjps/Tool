教室雙螢幕完美同步指南

拒絕無效操作，用最現實的邏輯解決教學現場的技術痛點

▍為什麼我們需要這個工具？

在教學現場，時間就是一切。我們沒有時間在講台上摸索 Windows 的設定、按右鍵找「顯示設定」，然後在延伸、複製之間猶豫不決。這不僅消耗老師的心力，也浪費學生的時間。

這份指南將引導您建立一個「一鍵執行檔」。只要點擊兩下，系統就會強制將教師機畫面與投影布幕（或大螢幕）完美同步（Clone Mode）。這是一個零容錯、防呆到極致的解決方案。

▍十分鐘部署：傻瓜式製作步驟

請老師們依照以下四個步驟執行，做過一次之後，您就可以將這個檔案複製到任何教室的電腦上使用。

1

顯示電腦的副檔名（極度重要）

Windows 預設會隱藏檔案的副檔名，這會導致我們無法正確建立執行檔。請打開任意資料夾，點擊上方的「檢視」標籤，並將「副檔名」的選項打勾。

2

在桌面上建立文字檔

回到桌面，在空白處點擊滑鼠右鍵 ➜ 選擇「新增」 ➜ 選擇「文字文件」。您可以先隨便命名，例如「螢幕同步」。

3

複製並貼上核心程式碼

打開剛剛新增的文字文件，將下方黑框內的程式碼全部複製並貼上到文字檔中，然後點擊「檔案」➜「儲存檔案」。

@echo off
:: ==========================================
:: 腳本名稱: 教學現場雙螢幕強制同步
:: 執行方式: 直接連點兩下執行
:: 核心邏輯: 呼叫 Windows 內建 DisplaySwitch
:: ==========================================

echo 正在強制將顯示器設定為「同步(複製)」模式...
echo 請稍候一秒鐘，螢幕可能會閃爍，這是正常現象。

%windir%\System32\DisplaySwitch.exe /clone

echo 執行完畢，您可以開始上課了。
exit
4

更改副檔名為 .bat（賦予執行能力）

關閉文字檔。在桌面上的該檔案點擊右鍵，選擇「重新命名」。將檔案名稱改為 一鍵同步螢幕.bat。系統會跳出警告問您是否確定要變更，請毫不猶豫地點選「是」。

▍如何使用與驗收

設定已經完成。現在，您的桌面上應該有一個名為「一鍵同步螢幕.bat」的檔案（圖示會變成帶有齒輪的視窗）。

實際使用方式：上課前，接上投影機線材，直接對著這個檔案連點兩下滑鼠左鍵。螢幕會短暫黑屏 1 到 2 秒，接著您的電腦畫面與投影布幕就會完全一模一樣。完美通關。

⚠️ 硬核除錯邏輯 (Troubleshooting)

如果您點了檔案卻沒有反應，請用最現實的角度檢查以下事項，不要懷疑程式碼，請懷疑硬體：

  • 線材真的有插好嗎？ 請重新拔插 HDMI 或 VGA 線，並確保投影機已經開機且切換到正確的訊號源。
  • 系統有抓到第二個螢幕嗎？ 如果電腦根本沒偵測到投影機，程式下達同步指令也無效。
  • 副檔名改對了嗎？ 確認檔案名稱結尾是 .bat，而不是 .bat.txt（這就是為什麼步驟 1 極度重要）。
